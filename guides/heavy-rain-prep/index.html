大雨・ゲリラ豪雨・台風のときに考えておきたい備え

更新：2026.06.30編集部

大雨やゲリラ豪雨、台風のときは、浸水・冠水・停電・断水・情報の途絶が重なって起こることがあります。いちばんの基本は「早めに安全な場所へ」。そのうえで、考えておきたい備えを場面ごとに整理します。

毎年、夏から秋にかけて大雨や台風の被害がくり返し起きています。短時間で道路が川のようになったり、河川の増水や内水であふれた水が建物に入りこんだり、広い範囲で停電したりと、複数の困りごとが同時に重なるのが特徴です。ここで何より大切なのは、気象庁や自治体の情報をこまめに確認し、危険を感じたら早めに安全な場所へ移ることです。道具はその判断を置きかえるものではなく、あくまで備えのひとつとして考えてください。

まず確認したいこと（情報と避難）

大雨のときは、雨雲の動き・注意報や警報・避難情報をこまめに確認することが基本になります。自分の住む場所が浸水や土砂災害のリスクのある区域かどうかは、ハザードマップポータルサイトなどで事前に確認しておくと、いざというときに動きやすくなります。停電や夜間に備えて、モバイルバッテリーを充電しておく、両手が空くヘッドライトを手の届くところに置いておく、といった準備もしておきたいところです。

浸水・車の水没に備える道具

自宅まわりへの水の入りこみが心配な場所では、止水板・吸水土のうを使う方法があります。ただしこれは入りこみを抑える備えのひとつで、浸水を防ぎ切るものではありません。設置の判断はハザードマップや自治体の指示を確認したうえで行い、早めの避難の判断が遅れないようにしてください。

車での移動中に冠水に出会ったときは、冠水した道路に入らない・無理に進まないことが基本です。やむを得ず車内に閉じ込められた場合に備える道具として緊急脱出ハンマーがありますが、万能ではなく使い方や限界の理解が前提になります。車の水没については別ページ「道路の冠水・車の水没で考えておきたい持ち物と、知っておきたいこと」もあわせてご覧ください。

停電・断線に備える道具

台風や落雷では、広い範囲で停電が起きることがあります。スマホの充電や、季節によっては扇風機・電気毛布まで考えるならポータブル電源が選択肢になります。冷蔵庫が止まったときの食品の一時的な保冷にはクーラーボックスと保冷剤、落雷による家電への影響が気になる場合は雷サージ対応の電源タップを検討する方法もあります。停電全般の備えは「停電時にあると便利な道具」で必要度の順に整理しています。

無理をしないための注意

大雨・台風の被害は、地域や状況によって大きく異なります。この記事は道具選びの参考として整理したもので、避難の判断や行動の安全を保証するものではありません。注意報・警報、避難情報、河川やダムの情報などは、気象庁・自治体・国土交通省などの公的機関の最新の情報を必ずご確認ください。水位が上がってからの移動は危険が増すため、明るいうち・水が増える前の早めの行動を心がけてください。

あわせて備えたい道具

止水板・吸水土のう →
大雨・河川氾濫・内水氾濫で室内への浸水の入りこみを抑えたいときに検討できる道具。状況により効果は異なるため、自治体・気象庁の情報と合わせて使う備えです。
緊急脱出ハンマー →
車のドアが開かないなど限られた状況で考えておく道具。使い方・限界・リスクを事前に把握し、公的機関の情報も合わせて確認しておくことが大切です。
ヘッドライト →
停電や夜間の移動で両手を使いながら灯りが必要なときに役立つ道具。明るさより「使いやすさ・電池の入手しやすさ」を優先して選ぶと長く使えます。
モバイルバッテリー →
スマホ・タブレットを外出先や停電時に充電したいときの基本道具。容量と充電速度を用途に合わせて選ぶと使いやすくなります。
防災ラジオ →
停電や通信障害で情報が得にくいときに、電池や手回しでラジオ放送を受け取るための道具。ライトや充電機能を兼ねるものも多く、停電・大雨の備えとして考えやすい一台です。
ポータブル電源 →
停電・車中泊・キャンプで電源を確保したいときに役立つ道具。スマホ充電中心なら大容量は必須ではありません。
クーラーボックス →
停電時に冷蔵庫が止まっても、食品の温度変化を抑えたいときに役立つ道具。保冷力と容量のバランスを用途に合わせて考えておくと選びやすくなります。
雷サージ対応 電源タップ →
落雷や急激な電圧変動による家電への影響を抑えたいときに検討できる道具。サージ吸収容量と雷ガード機能を確認して用途に合わせて選ぶのが基本です。
